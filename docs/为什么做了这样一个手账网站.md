
说来有点简单，又有点任性。

---

## 起因

一直想要一个**完全属于自己的小地方**。

社交平台太吵，算法总在猜我喜欢什么，评论区的噪音让人疲惫。  
我只想安安静静地写字，放几张喜欢的图，不用考虑流量和点赞。

于是决定：自己搭一个。

---

## 为什么是“手帐”风格？

手帐给人的感觉是**私人、随意、有温度**的。  
不需要精致的排版，不需要迎合谁，就像翻开一本写满字的本子，里面有涂鸦、有贴纸、有歪歪扭扭的字迹。

我希望这个网站也有这样的气质。

---

## 技术上的任性

我用了一个很简单的方案：  
- 写 Markdown 文件放在 `/docs` 里  
- 一个 HTML 文件把它们变成网页  
- 部署在 GitHub Pages，免费又省心  

没有复杂的数据库，没有后台管理，**想写的时候就新建一个 .md 文件**。

这种“原始感”反而让我更愿意动笔。

---

就这样，  
柑音手帐，  
慢慢写下去。
